+ + + THIS is VERSION 1.0-ALPHA + + + This version of Olex2 may be unstable. It is our experimental version that we use to test new features. You are welcome to to use this version, but please be aware that things will frequently not quite work out the way you might expect. Major Roadworks at the GUI junction Many parts of the Olex2 GUI have been written a long time ago and need to be renewed. This is happening now. So things on on the GUI will often appear broken, shifted, fragmented and whatever else you can imagine. Automatic Structure Inversion: OFF Olex2 will no longer invert the structure if it thinks it should be done so. Instead, there will be a warning. Postscript Olex ADP Plots ORTEP-style drawings can now be created straight from within Olex2. This new tool can be found under Tools>Images.
